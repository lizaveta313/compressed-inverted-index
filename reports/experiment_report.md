# Отчёт по экспериментальному тестированию

## Цель

Проверить работу обычного и сжатого инвертированного индекса, сравнить объём хранения, время построения и среднее время поиска. Отдельно проверялись запросы «Ректор СПбГУ» и «Ректор МГУ».

## Данные

Использовались два набора данных.

Первый набор - реальные документы из второго модуля:

```text
data/module2/spbu_documents.jsonl
```

Файл содержит 50 документов в формате JSONL. Текст берётся из поля `text`. При загрузке исправляется проблема кодировки, чтобы запросы нормальной кириллицей находились корректно.

Второй набор - сгенерированные документы для проверки масштабирования до 40 000 документов. Они создаются функцией `generate_documents(count, seed=42)` и не сохраняются как отдельный файл. Такой набор нужен, потому что в реальном файле второго модуля сейчас только 50 документов.

## Методика

- строился обычный инвертированный индекс;
- строился сжатый индекс с дельта-кодированием и гамма-кодом Элиаса;
- сравнивался объём хранения;
- измерялось время построения через `time.perf_counter()`;
- измерялось среднее время поиска по нескольким повторам;
- проверялось совпадение результатов поиска.

## Анализ

На реальных данных второго модуля сжатый индекс уменьшил объём хранения на 40.01% при максимальном доступном корпусе 50 документов. На сгенерированном корпусе из 40 000 документов экономия составила 90.90%.

При росте числа документов размер индекса растёт. Сжатый индекс занимает меньше места, потому что вместо полных `doc_id` сохраняются разности между ними, закодированные гамма-кодом Элиаса. Поиск в сжатом индексе медленнее, потому что перед пересечением postings lists их нужно декодировать.

Результаты поиска обычного и сжатого индекса совпали на всех проверенных запросах.

## Вывод

Инвертированный индекс реализован. Сжатие на основе дельт и гамма-кода Элиаса работает. Объём хранения уменьшается как на данных второго модуля, так и на большом сгенерированном корпусе. Поиск по «Ректор СПбГУ» и «Ректор МГУ» работает, результаты обычного и сжатого индекса совпадают.
